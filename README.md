**VeriTrust AI** is a frontend-based prototype designed to improve trust in e-commerce platforms by ensuring that only genuine buyers can post product reviews. It addresses the problem of fake and misleading reviews by validating user details against a simulated purchase database. If the entered information matches a valid record, the review is accepted; otherwise, it is rejected. The system also includes a simulated AI-based analysis to evaluate the authenticity of the review and generate a trust score. The workflow includes product selection, review submission, purchase verification, AI analysis, and final review display with an audit log for transparency. Built using HTML, CSS, and JavaScript, this lightweight and interactive prototype demonstrates how verified purchase validation and intelligent analysis can enhance reliability and user confidence in online shopping systems.
